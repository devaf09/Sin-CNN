正弦波干渉型のニューラルネット構想



コンセプト

　AIのニューラルネットワークの畳み込み層を、物理的な「波の干渉」と「回折」の原理に基づいた正弦波(sin関数)の集まりで構成するもの。

目的



・スケール自在：sin関数ベースの畳み込み層とし、解像度等に依存しない汎用的な認知モデルを構築する。



・再学習コストの低減：学習済みの干渉波パターン層群（以下パッケージ）の融合を、複素数としての乗算（位相の加算）により行う。



・AIの認知のブラックボックス化の回避：干渉波パターンによる追跡と、学習済みパッケージのモジュール化、分化を行う。



・UNIXパス型の抽象化：みかん/ 丸いの反応.map だいだい色の反応.map 未知の要素反応Xa.map  未知の要素反応Xb.map





概要



・sin波の干渉を効率よく計算するため、複素数による計算に絞る。







Sin型CNNの効果



・既知の要素（ラベル化された特徴）と未知の要素（潜在的な特徴、ラベル化困難な要素）を、同時にパターン化、認識できる。



・効率的な情報の重なり：N次元空間では１つの層に膨大な数の「直交するsin波」を内蔵できる。



・学習コストの低減：現状のCNNでは次元増による計算量の増加が問題だが、正弦波の場合はベクトルの内積を計算するだけなので、次元が増えても計算負荷の増加を抑えられる。



・非局所性の相関：離れた点、又は離れた次元同士の点の干渉を容易に計算できる。



・ある種のシード値化が可能となり巨大な重みデータを保存しなくても層再現が容易となる。



・sin関数は連続的な数式なので、入力の解像度が変わっても再計算で滑らかに適応できる。（現実的な計算量で宇宙を飲み込めるのが望ましい。）





アプローチ

Step 001: 文字認識（MNIST等）を用いた、干渉波による特徴抽出の妥当性検証。



Step 002: 複素数バックプロパゲーションの実装と、連続体入力に対する精度検証。











ちょっと説明

　文字での説明ではわかりにくいので、たとえ話で説明します！



　既知のニューラルネットワークの各層は１次元的な数列からランダムな波形と見なすことができます。

　すべての波形パターンはsin波の組み合わせで構成できるので、これは最終的にフーリエ展開(FSE)が可能です。（複雑であっても数学的に整理が可能です。）

　

　しかし、現在のニューラルネットワークの学習は重みの計算を全量行うため効率が悪く、フーリエ級数展開（以下、FSE）にはランダム過ぎます。

　なので、最初から干渉パターンとしてのニューラルネットワークが可能か検討、実際にコード化して検証することが目的です。

　思考実験的な概念での説明ですが、下記の方法で、sin波による干渉波でのニューラルネットワークに切り替えます。

　

　イメージの説明：画像認識の場合。

　　物理学的な光の散乱と集光をイメージしてお読みください。

&#x09;１．例えばみかんの画像を認識しようとする際、みかんの画像データは数列なので、複数の周波数成分を持った波形データとして変換できます。これを２次平面の光スペクトルと考えます。（以下みかんの光）

&#x09;２．次に複数の波紋を持つガラス板にみかんの光を通します、これにより光の回折と干渉が起きます。（これがCNNで言う層にあたります。）

&#x09;３．適度に通過する光を増幅させながら、複数の層を通過すると特定の場所で同位相の部分が明るくなります。（集光されます。）すなわち複数の集光点＝反応点が発生します。

&#x09;４．このガラスの板の波紋、すなわちsin関数のパラメータを複数のみかんの画像で同じような集光する部分を作るよう学習させます。最終的にみかんを示す複数の集光点のマップを作ります。

&#x09;５．このマップは、イメージではぼんやりとした光の点を想像されるかもしれませんが、実際にカオス論的な光の散乱が起り、画像のわずかな変化、例えば光の周波数が変わると干渉波の焦点も変わるため、１層では僅かなズレであっても、複数の層を通過するうち集光点のズレは指数関数的ズレていきます。（二重振り子のような、初期値わずかな差に対する鋭敏感を持ちます。）

&#x09;６．このため、学習後のガラス層群は、複数のみかんの光が持つ共通しない反応の部分をその過程ではじき出し、逆に共通するみかん感に対する反応は鋭くなります。

&#x09;７．この時、複数のみかんを示す集光点はみかんの画像の要素を含むものになります、すなわち、丸い部分を示す光の点、橙色を示す光の点、といったみかんの要素の点の集合になります。が、これは人がみかんを認識するのと同様「これはみかん」という認識になるため、概念化や言語化ができる要素（丸いの反応、だいだい色の反応）と、未知の要素（みかん要素の反応Xa、Xb）に分かれます。つまり、一部分では人では説明が難しい認知外のみかん要素を発見することになります。（なんとなくみかんを含む）

&#x09;　　これによりAIのブラックボックス化の回避、学習の要点認識、（人と比較して）認識の拡張が可能です。

&#x09;８．sin波は無限に続くため、入力に対しスケールを示す倍音や数列を追加することで入力サイズが広がっても、計算のみで拡張が可能です。惑星サイズのみかん画像であっても、同様に認識します。

&#x09;９．作成したみかんの学習パターンは周波数成分のみで出来ているため、他の学習パターン、例えばトマトの学習パターンと融合(乗算)が可能です、実際には複素透過率の積としてコード化できます。（すなわちみかんとトマトを再学習が不要です。）

&#x09;　　新しく作ったみかんとトマトの学習パターンのガラス層群は、それぞれの画像において、みかんとトマトを示す集光点をつくります。　また、cat input | filter\_mikan | filter\_tomato | output\_map のようにも行えます。

&#x09;１０．これらは、従来の「重みを書けて足す」CNNモデルとは異なり、「物理をシミュレーション」するに近いため計算効率が良く、情報の欠落を少なくすることが可能と考えます。また、今後発展する光演算チップとの相性も良いと想定します。
